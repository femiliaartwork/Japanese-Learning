N5
中　ー　音読み：ちゅう　e.g. 中国
　　　　　訓読み：　なか　e.g. 部屋の中

長　ー　音読み：ちょう　　e.g. 校長先生
　　　　　訓読み：ながい e.g. 長い髪

出　ー　音読み：しゅつ　e.g. 出発（しゅっぱつ）
　　　　　訓読み：で（る）　e.g. 外に出る
　　　　　　　　　　だ（す）　e.g. 手を出す

時　ー　音読み：じ　e.g. 一時
　　　　　訓読み：とき e.g. 時々/ 子供の時

行　ー　音読み：こう e.g. 旅行
　　　　　　　　　　ぎょう　e.g. 行事 (event)
　　　　　訓読み：い　e.g. 行く
　　　　　　　　　　おこな e.g. 祭りを行う (hold an event)

見　－　音読み：けん　e.g. 意見
　　　　　訓読み：み（る） e.g. 見る　見える　見せる　

月　ー　音読み：　げつ　e.g. 月曜日
　　　　　　　　　　　がつ　e.g. 一月
　　　　　訓読み：　つき　e.g. 月が綺麗

分　ー　音読み：　ぶ　e.g. 半分
	　　　　ぷん e.g.　三分
　　　　　訓読み：　わ　e.g. 意味が分かる

後　ー　音読み：　ご　e.g. 午後　／　最後
　　　　　訓読み：　うし　e.g. 後ろ　
　　　　　　　　　　　あと　e.g その後　

前　－　音読み：ぜん　e.g. 前回　午前
　　　　　訓読み：まえ　e.g. 一時間前　／　家の前

生　ー　音読み：せい　e.g. 先生　／　学生
	　　　しょう　e.g 一生
　　　　　訓読み：い　　e.g. 生きる
	     う　 e.g 生まれる
　　　　　　　　　　は　　e.g. 生える
　　　　　　　　　　なま　e.g. 生野菜

間　ー　音読み：かん　e.g. 時間
　　　　　　　　　　けん　e.g. 人間
　　　　　訓読み：あいだ　e.g. 二人の間　/ この間
　　　　　　　　　　ま　e.g. 間に合う

上　ー　音読み：じょう　e.g. 以上／上手
　　　　　訓読み：うえ　e.g. 机の上（つくえのうえ）
	　　　　あ（がる）　e.g. ねだんが上がる／エレベーターが上がる
	　　　　あ（げる）　e.g. 手を上げる／ボリュームを上げる

今　ー　音読み：こん　e.g. 今週　
　　　　　　　　　　きょう　e.g. 今日は月曜日
　　　　　訓読み：いま　e.g. 今、行きます

金　ー　音読み：きん　e.g. 金曜日／料金／現金
　　　　　訓読み：かね　e.g. お金／金持ち

入　ー　音読み：にゅう　e.g. 入学
　　　　　訓読み：はい（る）　e.g. 部屋に入る
　　　　　　　　　　い（れる） e.g. 水を入れる
